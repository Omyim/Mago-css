การใช้งาน Mago CSS พัฒนามาจากตัว Slimer CSS

Mago คือไลบรารีย์ที่ถูกพัฒนาขึ้นเพื่อการใช้งานและความเหมาะสมกับหน้าจอของหน้าจอใหญ่ๆ เช่น หน้าจอคอมพิวเตอร์ โน๊ตบุ้ค แม็ค ครับและหน้าที่ท่านกำลังรับชมอยู่ก็คือหน้าที่ทำจาก Mago ทั้งหมดไม่มีการปรับแต่งอะไรทั้งสิ้น เราทำคอมโนเนนท์ออกมาทั้งหมด 7 ตัวครับซึ่งเราได้ทำการแยกไฟล์ไว้ทั้งหมด 10 ไฟล์เพื่อง่ายต่อการแก้ไขและอัพเดทงาน และรวมน้ำหนักไฟล์ทั้งหมดเพียง 19.7 kb ครับ ซึ่งน้อยมากจากตัว Slimer CSS หากเปรียบเทียบความทันสมัยและความใหม่แล้วผมให้ Mago มากกว่าตัว Slimer CSS ครับในขณะนี้ ตัวนี้ขึ้นอยู่ที่เวอร์ชั่น 1
