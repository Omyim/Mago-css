Component

ตัวเลือกสำหรับการใช้งาน ตัวเลือกสำหรับการใช้งานมีไม่ค่อยเยอะเท่าไหร่ เนื่องจากไม่ค่อยมีเวลาพัฒนา การใช้งานเราแนะนำให้ท่านได้ใช้ตัวเทมเพลตของเราก่อนที่จะลงมือสร้างเว็บของท่านครับ 
<!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="UTF-8">
	<title>Mago Template 101</title>

	<!-- Mago Stylesheet -->
	<link rel="stylesheet" href="css/mago.min.css">

	<!-- HTML5Shiv -->
	<script type="text/javascript" src="../js/html5shiv.min.js"></script>
	<script type="text/javascript" src="../js/html5shiv-printshiv.min.js"></script>
</head>
<body>
	<h1>Hello, World !</h1>
</body>
</html>

Forms

ฟอร์มและปุ่มก็เป็นสิ่งหนึ่งที่สำคัญในการสร้างเว็บไซต์ ดังนั้นเราจึงต้องทำให้ออกมาดุดีและรองรับหลายบราวเซอร์ให้มากที่สุดเท่าที่ดีจะทำได้ได้ครับ การเรียกใช้งานคลาสก็เหมือนกับ ของ Bootstrap 3 คือ .form-group .form-control

Example

Usage 

<form>
	<div class="form-group">
		<label for="example-textbox-1">Username : </label>
		<input type="text" id="example-textbox-1" class="form-control" placeholder="Username ...">
	</div>
	<div class="form-group">
		<label for="example-textbox-2">Password : </label>
		<input type="password" id="example-textbox-2" class="form-control" placeholder="Password ...">
	</div>
	<div class="form-group">
		<input type="submit" class="button th-primary" value="Submit">
		<input type="reset" class="button th-danger" value="Reset">
	</div>
</form>

Addon in input

Addon คือสิ่งที่เพิ่มมาเรื่อยสามารถแก้ไขได้ตลอด Addon ในฟอร์มสามารถใช้งานได้อย่างง่ายๆ ให้ลองสังเกตุจากคลาสดูครับ

Example

Country

ในฟอร์มจะพบว่ามีคำนำหน้าสามารถใช้แทน Label ได้เลยและสวยงามด้วย เราใช้คลาส .form-group addon
Usage 

<div class="form-group addon">
	<label>Country</label>
	<div class="input-addon">
		<span>Country</span>
		<select class="form-control">
			<option>Select Country..</option>
			<option>Thailand</option>
			<option>Laos</option>
		</select>
	</div>
</div>


Buttons

ตัวเลือกสำหรับการใช้งานปุ่มของเรามีอยู่ เท่ากันกับ Bootstrap นั่นก็หมายความว่ามีทั้งหมด 6 รูปแบบ การเรียกใช้งานรูปแบบทั้งหมดจะต้องเรียกใช้งานคลาส .button ครับเหากใครต้องการใช้งานรูปแบบปกติเรียกใช้งานแบบนี้ครับ .button หากจะเรียกใช้รูปแบบอื่นต้องใส่เพิ่มจากเก่ามาเป็น .button th-ชื่อรูปแบบ

Example

Usage 

<button type="button" class="button">Default</button>
<button type="button" class="button th-primary">Primary</button>
<button type="button" class="button th-info">Info</button>
<button type="button" class="button th-success">Success</button>
<button type="button" class="button th-danger">Danger</button>
<button type="button" class="button th-warning">Warning</button>

Breadcrumb

Breadcrumb คือการบอกตำแหน่งของหน้าเพื่อให้รู้ว่าตอนนี้ท่านกำลังรับชมอยู่ที่หน้าไหนประเภทอะไร ทั้งนี้ Breadcrumb เราทำออกมาสองรูปแบบครับ .breacrumb (ปกติสีออกเทา) .breadcrumb .breadcrumb-inverse (รูปแบบที่สองสีดำอ่อน)

Example

หน้าหลัก หน้าที่สอง

รูปแบบที่สอง
หน้าหลัก หน้าที่สอง

จากตัวอย่างท่านจะสังเกตุพบว่ามันมีจุดเด่นที่คำว่า หน้าที่สอง เนื่องจากเราได้ทำการใส่คลาส .active เข้าไปใน Element

Usage 

<div class="breadcrumb">
	<a href="JavaScript:void(0);"><span>หน้าหลัก</span></a>
	<a href="JavaScript:void(0);" class="active"><span>หน้าที่สอง</span></a>
</div>

<-- แบบที่สอง -->
<div class="breadcrumb breadcrumb-inverse">
	<a href="JavaScript:void(0);"><span>หน้าหลัก</span></a>
	<a href="JavaScript:void(0);" class="active"><span>หน้าที่สอง</span></a>
</div>
